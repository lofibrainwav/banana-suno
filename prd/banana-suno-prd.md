# Banana Suno PRD

본 프로젝트의 주요 목표는 다음과 같습니다.

1. **Apify Actor와 Banana AI 통합**
   - Apify 기반 스크래핑/자동화 로직을 Banana AI 플랫폼과 연동합니다.
2. **안전한 MCP 도구 실행 환경 제공**
   - `mcp-safety-recovery.js` 등을 통해 도구 실행 중 발생할 수 있는 오류를 복구하고 로그를 관리합니다.
3. **모듈화된 구조와 테스트 강화**
   - `src/` 디렉터리의 모듈화된 코드와 Jest 테스트를 통해 유지보수성을 높입니다.
4. **Node.js 18+ 및 TypeScript 지원 계획**
   - 최신 런타임을 사용하고 단계적으로 TypeScript 마이그레이션을 진행합니다.
5. **예제 스크립트 제공**
   - `main-refactored.js`, `banana-ai-mcp-integration.js`와 같은 스크립트를 통해 사용 방법을 제시합니다.
